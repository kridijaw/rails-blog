## Requirement

- Ruby `3.2.2`
- Rails `7.1.3`
- sqlite3 `1.4`

## Install Dependencies (First time only)

- Install ruby `3.2.2`
- Install rails dependencies: `bundle install`

## Database creation & initialization

- `bin/rails db:create`
- `bin/rails db:migrate`
